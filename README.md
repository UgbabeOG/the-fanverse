# Pacheco Team Fan Hub

This is the official fan community for Isiah Pacheco, built with Next.js and Firebase Studio.

## Features

- **Homepage:** A welcoming hero section with CTAs to explore highlights and join the community.
- **Highlights:** A blog section to view the latest news and updates.
- **Membership:** Exclusive membership tiers for dedicated fans.
- **Merchandise:** A page to showcase official team gear.
- **About:** Learn more about the mission and team behind the community.
- **Contact:** Get in touch with Team Pacheco.

This project was bootstrapped with Firebase Studio.
